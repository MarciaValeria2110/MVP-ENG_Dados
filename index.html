Telefonia Móvel 4G e 5G no Brasil

Data Warehouse & Analytics Project (BigQuery)

GitHub Code BigQuery Notebook

Project Overview

Este projeto constrói um Data Warehouse multidimensional para analisar a cobertura da telefonia móvel 4G e 5G nos municípios brasileiros, integrando dados de presença e Qualidade de Experiência (QoE).

  • Role: Data Engineer / Data Analyst
  • Focus: Data Warehouse · Telecom · Analytics
  • Stack: BigQuery · SQL · GCP

Key Findings

  • Desigualdade regional significativa na cobertura 5G
  • Estratégias distintas de expansão entre operadoras
  • QoE superior em 5G, porém com menor volume de testes
  • Lacunas críticas de cobertura em municípios periféricos

Methodology

  1. Ingestão de dados públicos (CSV) via GCS
  2. Camada Bronze–Silver–Gold
  3. Modelagem Estrela com PK/FK naturais
  4. Views analíticas para BI

Tech Stack

BigQuery · Google Cloud Storage · SQL · GitHub Pages

Impact & Use Cases

  • Identificação de lacunas de infraestrutura digital
  • Suporte a políticas públicas e regulação
  • Análise estratégica de mercado de telecom

Data Source: Dados públicos (ANATEL / IBGE)
